REST

RE - Representational

S - State

T - Transfer


NO Rules to form HTTP Request / Response (unlike SOAP)

Used to complete a Web Service (API) using HTTP

Request Parts:

– Start Line All Methods (GET, POST, PUT, DELETE ...)

– Header Line All

– Blank Line

– Body ALL (JSON, XML, images, html ...)
